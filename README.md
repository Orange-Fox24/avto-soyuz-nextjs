## О проекте

Дипломный проект по специальности 09.02.07 «Информационные системы и программирование».  
Тема: **«Разработка и внедрение информационной системы для автоматизации управления логистической компанией ООО "АВТО-СОЮЗ"»**

Веб-приложение для автоматизации управления заказами, отслеживания грузов и взаимодействия между клиентами, менеджерами и водителями логистической компании.

## Ролевая модель

| Роль | Возможности |
|------|-------------|
| **Гость** | Просмотр информации о компании, услуг, тарифов, контактов. Отправка заявок. Авторизация. |
| **Клиент** | Личный кабинет, создание заказов, отслеживание статуса, история заказов, редактирование профиля |
| **Менеджер** | Панель управления: просмотр всех заказов, назначение водителей, изменение статусов, отчёты, список водителей |
| **Водитель** | Интерфейс водителя: просмотр назначенных рейсов, изменение статуса выполнения заказа |

## Технологический стек

| Категория | Технология |
|-----------|------------|
| **Фреймворк** | Next.js 15 (App Router) |
| **Язык** | TypeScript |
| **База данных** | PostgreSQL |
| **ORM** | Prisma |
| **Аутентификация** | NextAuth.js (Credentials Provider + JWT) |
| **Стилизация** | CSS Modules |
| **Шрифты** | Actay, Actay Wide, Wadik (собственные) |
| **Деплой** | Vercel |

## Аутентификация и авторизация

- **NextAuth.js** с провайдером Credentials (email/пароль)
- **JWT-токены** хранятся в HTTP-only cookies
- **Ролевая модель**: CLIENT, MANAGER, DRIVER
- **Middleware** защищает маршруты по ролям
- Пароли хешируются через **bcrypt** (12 раундов)

## Основные функции

### Клиент:
- Создание заказа (город отправки/назначения, вес, тип груза, дата, комментарий)
- Отслеживание статуса заказа (Новая → Назначен → Погрузка → В пути → Доставлен)
- История заказов
- Редактирование профиля

### Менеджер:
- Просмотр всех заказов компании
- Назначение водителя на заказ
- Изменение статуса заказа
- Список водителей с занятостью
- Модуль отчётов (статистика, последние заказы, топ водителей)

### Водитель:
- Список назначенных рейсов
- Пошаговое изменение статуса (Погрузка → В пути → Доставлен)

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/Orange-Fox24/avto-soyuz-nextjs

# Перейти в папку проекта
cd avto-soyuz-nextjs

# Установить зависимости
npm install

# Настроить переменные окружения
cp .env.example .env.local
# Отредактировать .env.local: DATABASE_URL, AUTH_SECRET

# Применить миграции БД
npx prisma db push
npx prisma generate

# Запустить dev-сервер
npm run dev
