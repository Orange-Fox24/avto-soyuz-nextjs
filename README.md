## О проекте

Дипломный проект по специальности 09.02.07 «Информационные системы и программирование».  
Тема: **«Разработка и внедрение информационной системы для автоматизации управления логистической компанией ООО "АВТО-СОЮЗ"»**

Веб-приложение для автоматизации управления заказами, отслеживания грузов и взаимодействия между клиентами, менеджерами и водителями логистической компании.

## Ролевая модель

| Роль | Возможности |
|------|-------------|
| **Гость** | Публичный сайт, калькулятор стоимости, формы заявок, скачивание документов, FAQ |
| **Клиент** | Личный кабинет, создание заказов с автоподсказками, отслеживание статуса, отмена, профиль |
| **Менеджер** | Все заказы с фильтрами, назначение водителя, установка цены, оплата, отчёты PDF, водители (отпуск), транспорт, заявки |
| **Водитель** | Активные рейсы и история, смена статуса с подтверждением, карта маршрута, кнопка «Проблема», профиль |

## Технологический стек

| Категория | Технология |
|-----------|------------|
| **Фреймворк** | Next.js 15 (App Router) |
| **Язык** | TypeScript |
| **База данных** | PostgreSQL |
| **ORM** | Prisma |
| **Аутентификация** | NextAuth.js (Credentials + JWT) |
| **Стилизация** | CSS Modules |
| **Шрифты** | Actay, Actay Wide, Wadik |
| **Деплой** | Vercel + Neon |

## Аутентификация и безопасность

- NextAuth.js с провайдером Credentials (email/пароль)
- JWT-токены в HTTP-only cookies
- Ролевая модель: CLIENT, MANAGER, DRIVER
- Middleware защита маршрутов
- Пароли: bcrypt (12 раундов)
- Защита от SQL-инъекций: Prisma ORM
- Валидация пароля: 8+ символов, заглавная, строчная, цифра, спецсимвол
- Маска телефона +7

## Основные функции

### Гость:
- Публичный сайт с информацией об услугах, тарифах, контактах
- Калькулятор стоимости с учётом расстояния, веса и доп. услуг
- Формы заявок (перевозка, обратная связь, партнёры)
- Скачивание документов (PDF)
- FAQ с раскрывающимися ответами

### Клиент:
- Создание заказа с автоподсказками городов
- Автоматический расчёт примерной цены
- Отслеживание статуса (Новая → Назначен → Погрузка → В пути → Доставлен)
- Детали заказа, отмена
- Профиль (имя, телефон)

### Менеджер:
- Все заказы с фильтрами по статусу и дате
- Выпадающее меню действий: назначение, цена, оплата, статус
- Однократная установка цены и оплаты (защита от изменений)
- Водители: статус занятости, отпуск
- Транспорт: добавление, доступность
- Заявки с публичных форм
- Отчёты с экспортом в PDF

### Водитель:
- Активные рейсы и история завершённых
- Смена статуса: Погрузка → В пути → Доставлен (с подтверждением)
- Карта маршрута по реальным дорогам (OSRM + Leaflet)
- Кнопка «Проблема» (поломка, ДТП, опоздание)
- Профиль водителя

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/Orange-Fox24/avto-soyuz-nextjs

# Перейти в папку проекта
cd avto-soyuz-nextjs

# Установить зависимости
npm install

# Настроить переменные окружения
cp .env.example .env.local
# Отредактировать .env.local: DATABASE_URL, AUTH_SECRET

# Применить миграции БД
npx prisma db push
npx prisma generate

# Запустить dev-сервер
npm run dev
